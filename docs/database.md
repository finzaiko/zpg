# Database

>On progress..

Show all database in connected server

## Create Connection
### 1. Click Connection Icon [db-icon]
### 2. Click Add button
### 3. Input connection
- **Name**: Connection name
- **Host**: Host DB server
- **Port**: Port DB server
- **Username**: Username database
- **Password**: Password database
### 4. Test Connection (optional)
### 5. Click **Save** connection
